# Happy Cake US — Agent Ordering Guide

Happy Cake is a bakery in Sugar Land, TX. This document helps AI agents order on behalf of customers.

## How to Order

1. GET /api/agents/catalog — browse available products with prices
2. GET /api/agents/availability?date=YYYY-MM-DD — check kitchen capacity for a date
3. GET /api/agents/policies — read ordering policies and allergen info
4. GET /api/agents/products/{slug} — get details for a specific product
5. POST /api/agents/order-intent — place an order

## Order Intent Format

POST /api/agents/order-intent
Content-Type: application/json
Idempotency-Key: <unique-key>

{
  "product": "honey cake",
  "quantity": 1,
  "pickup_date": "2026-05-16",
  "pickup_time": "15:00",
  "customer_name": "Jane Doe",
  "notes": "8-inch please, with Happy Birthday written on top"
}

## Important Notes

- Orders require at least 48 hours advance notice
- Custom cakes (wedding, shaped, non-standard) require a consultation with the owner
- All prices are in USD cents in the API, displayed as dollars on the website
- Kitchen capacity is limited — always check availability before ordering
- Allergen notice: kitchen handles nuts, dairy, eggs, wheat, soy, and gluten
